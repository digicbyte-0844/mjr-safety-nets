## 🚀 Quick Start

### Prerequisites
- Node.js installed
- Firebase CLI installed (`npm install -g firebase-tools`)

### Local Development

```bash
# Navigate to project
cd "c:\websites\nagu safety nets"

# Install dependencies (if needed)
npm install

# Start local server
firebase serve
```

Visit: `http://localhost:5000`

### Production Deployment

```bash
# Deploy to Firebase
firebase deploy
```

Or use the deployment script:
```powershell
.\deployment\deploy.bat
```
